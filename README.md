## 📱 이상형 MBTI 테스트 앱

mbti가 유행하던 당시, 나의 이상형은 어떤 유형의 mbti일까? </br> 고민하게 되며 
이 앱을 개발하였습니다.

---

## ✨ 주요 기능

- 회원가입 & 로그인
- 본인의 MBTI 입력 및 저장
- MBTI 테스트 진행
- 이상형의 MBTI 결과, 사용자와 이상형의 궁합 확인
